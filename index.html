Event Generator

Project Synopsis

Lenel S2 develops innovative security technology and progressive solutions that deliver leading integrated access control. To provide flexible and reliable options for their consumers, they are constantly improving existing products and features. In collaboration with Lenel S2, we seek to renovate an existing tool known as EventGenerator, a part of their main product, OnGuard. EventGenerator is a tool that simulates machine events for the purpose of testing and troubleshooting systems. The core project involves rewriting the existing functionality in C# and providing a mechanism for remote access in the form of a web application. The goal is for users to be able to more easily test and troubleshoot their systems with the web application, as opposed to the already existing EventGenerator tool that is a part of OnGuard.
